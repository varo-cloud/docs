Getting started with Varo Cloud takes less than a minute. There is no password to create — you sign in with your email address and a one-time verification code, and your account is created automatically the first time you sign in.

## Create your account

<Frame caption="The Varo Cloud sign-in page at varo.cloud/auth">
  <img src="/images/auth-sign-in.png" alt="Varo Cloud sign-in page showing email and verification code fields" />
</Frame>

To create your Varo Cloud account:

1. Navigate to [https://varo.cloud/auth](https://varo.cloud/auth) in your browser.
2. Complete the human verification check (powered by Cloudflare). It usually passes automatically.
3. Enter your email address and click **Get Code**.
4. Check your inbox for a 6-digit verification code from Varo Cloud. The code expires in 10 minutes — if it doesn't arrive, use **Resend code**.
5. Enter the code and click **Sign in**.

That's it — if this is your first time signing in, your account is created on the spot and you land directly in the Varo Cloud dashboard.

<Note>
  By signing in, you agree to the Varo Cloud [Terms of Service](https://varo.cloud/terms) and [Privacy Policy](https://varo.cloud/privacy). Google and GitHub sign-in are coming soon.
</Note>

## Around the dashboard

Once you're signed in, the navigation bar gives you access to everything:

| Section | What it's for |
|---|---|
| **Models** | Browse the catalog of video, image, audio, and language models available through the unified API. |
| **AI Generator** | Run models directly in the browser — no code required. Great for testing prompts and comparing models. |
| **Pricing** | Per-model pricing for every model in the catalog. |
| **API Keys** | Create and manage the keys that authenticate your API and SDK requests. |
| **Billing** | Top up your balance, manage payment methods, and review your transaction history. |
| **Generations** | Full history of your generation tasks across all models and channels. |

## Language

The dashboard is available in **English** and **简体中文**. Switch languages at any time using the language selector in the navigation bar.

<Tip>
  Once your account is set up, the next step is generating an API key so you can authenticate programmatic requests. See [Authentication](/setup/authentication) for a step-by-step guide.
</Tip>
